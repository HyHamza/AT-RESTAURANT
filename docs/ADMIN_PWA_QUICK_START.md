# Admin PWA Quick Start Guide

## For Administrators

### What is the Admin PWA?

The Admin PWA (Progressive Web App) is a standalone version of the AT Restaurant admin panel that you can install on your device. It works like a native app with:

- **Quick Access**: Launch directly from your home screen
- **Offline Support**: View cached data even without internet
- **Push Notifications**: Get real-time order alerts
- **Separate App**: Keeps admin tasks separate from customer app

### Installation Instructions

#### On Desktop (Chrome/Edge)

1. **Log in** to the admin panel at `yoursite.com/admin`
2. **Wait** for the install prompt to appear (bottom-right corner)
3. **Click** "Install Admin App" button
4. **Confirm** in the browser dialog
5. **Done!** The admin app will open in a new window

Alternative method:
- Click the install icon (⊕) in the browser address bar
- Select "Install AT Restaurant Admin"

#### On Android (Chrome)

1. **Open** the site in Chrome browser
2. **Log in** to admin panel
3. **Wait** for the install banner at the bottom
4. **Tap** "Install Admin App"
5. **Confirm** installation
6. **Find** the app icon on your home screen

#### On iOS (Safari)

1. **Open** the site in Safari browser
2. **Navigate** to `/admin` and log in
3. **Tap** the Share button (⬆️) at the bottom
4. **Scroll down** and tap "Add to Home Screen"
5. **Name** it "AT Admin" (or your preference)
6. **Tap** "Add"
7. **Find** the app icon on your home screen

### Using Both Apps

You can install both the customer app and admin app:

- **Customer App**: For browsing menu and placing orders
- **Admin App**: For managing the restaurant

Each app:
- Has its own icon on your device
- Opens to its respective section
- Maintains separate login sessions
- Works independently

### Features

#### When Online
- Real-time order notifications
- Live data updates
- Full admin functionality
- Image uploads

#### When Offline
- View cached orders
- Browse menu items
- View customer data
- Access dashboard stats

### Troubleshooting

#### Install prompt doesn't appear?

**Try:**
1. Refresh the page
2. Clear browser cache
3. Check you're using HTTPS
4. Try the browser's install option (address bar icon)

#### App won't install?

**Check:**
- You're logged in as admin
- Using a supported browser (Chrome, Edge, Safari)
- Site is served over HTTPS
- No browser extensions blocking installation

#### App opens to wrong page?

**Solution:**
- Uninstall the app
- Clear browser data
- Reinstall following the steps above

### Uninstalling

#### Desktop
1. Right-click the app icon
2. Select "Uninstall" or "Remove"

#### Android
1. Long-press the app icon
2. Drag to "Uninstall" or tap app info
3. Select "Uninstall"

#### iOS
1. Long-press the app icon
2. Tap "Remove App"
3. Confirm "Delete App"

### Tips

1. **Install on all devices**: Install on phone, tablet, and desktop for maximum flexibility
2. **Enable notifications**: Allow notifications for real-time order alerts
3. **Keep both apps**: Install both customer and admin apps for complete access
4. **Update regularly**: The app updates automatically when you open it
5. **Bookmark admin login**: Save admin URL for quick access before installation

### Support

If you encounter issues:
1. Check the browser console for errors
2. Verify your admin privileges
3. Ensure stable internet connection
4. Try a different browser
5. Contact technical support

### Security Notes

- Each app maintains its own login session
- Admin app requires authentication
- Sessions expire after inactivity
- Always log out on shared devices
- Use strong passwords for admin accounts
